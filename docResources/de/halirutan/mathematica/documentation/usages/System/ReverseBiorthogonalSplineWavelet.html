ReverseBiorthogonalSplineWavelet

  • ReverseBiorthogonalSplineWavelet[] represents a reverse biorthogonal spline wavelet of order 4 and dual order 2.
  • ReverseBiorthogonalSplineWavelet[n,m] represents a reverse biorthogonal spline wavelet of order n and dual order m.

Attributes: Protected, ReadProtected
