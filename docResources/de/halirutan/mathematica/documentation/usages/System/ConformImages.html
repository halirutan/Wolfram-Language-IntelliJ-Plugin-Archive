ConformImages

  • ConformImages[{image1,image2, ... }] returns a list of images where all imagei are made to have conforming properties, including dimensions, data type, color space, and interleaving.
  • ConformImages[{image1,image2, ... },spec] returns all images of the specified spec.
  • ConformImages[{image1,image2, ... },spec,fitting] resizes images using the specified fitting method.

Attributes:Protected, ReadProtected

Options: AlignmentPoint, Axes, AxesEdge, AxesLabel, AxesOrigin, AxesStyle, Background, BaselinePosition, Boxed, BoxRatios, BoxStyle, ClipPlanes, ClipPlanesStyle, ClipRange, ColorFunction, ColorSpace, CoordinatesToolOptions, ImageResolution, ImageSize, Interleaving, Magnification, MetaInformation, Method, Padding, RotationAction, SphericalRegion, TaggingRules, ViewAngle, ViewCenter, ViewMatrix, ViewPoint, ViewRange, ViewVector, ViewVertical
