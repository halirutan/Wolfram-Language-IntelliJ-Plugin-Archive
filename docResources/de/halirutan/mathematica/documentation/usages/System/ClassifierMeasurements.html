ClassifierMeasurements

  • ClassifierMeasurements[classifier,testset,prop] gives measurements associated with property prop when classifier is evaluated on testset.
  • ClassifierMeasurements[classifier,testset] yields a ClassifierMeasurements[ ... ] object that can be applied to any property. When repeated property lookups are required, this is typically more efficient.

Attributes:Protected, ReadProtected

Symbol has no options.
