ChannelObject

  • ChannelObject[] gives a new anonymous channel specification.
  • ChannelObject["https:// … "] represents a channel specified by a given URI.
  • ChannelObject["relpath"] represents a channel for the currently authenticated user at a relative path.
  • ChannelObject["id"] represents the home channel of the user with Wolfram ID id.
  • ChannelObject["id, TI]:StyleBox[path"] represents a channel for the user with the specified ID at the given path.
  • ChannelObject[/StyleBox[abspath, TI]] represents a channel at an absolute path on the channel broker.

Attributes:Protected, ReadProtected

Symbol has no options.
