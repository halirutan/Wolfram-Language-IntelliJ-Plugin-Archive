Classify

  • Classify[{example1 → class1,example2 → class2, ... }] generates a ClassifierFunction[ ... ] based on the examples and classes given.
  • Classify[{example1,example2, ... } → {class1,class2, ... }] generates the same result.
  • Classify[ <| class1 → {example11,example12, ... },class2 → {example21, ... }, ... |> ] generates a ClassifierFunction[ ... ] based on an association of classes with their examples.
  • Classify[training,data] attempts to classify data using a classifier function deduced from the training set given.
  • Classify["name",data] attempts to classify data using the built-in classifier function represented by "name".
  • Classify["name",data,prop] gives the specified property of the classification associated with data.

Attributes:Protected, ReadProtected

Symbol has no options.
