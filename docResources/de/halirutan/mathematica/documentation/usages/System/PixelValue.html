PixelValue

  • PixelValue[image,{x,y}] gives the pixel value of image at position {x,y}.
  • PixelValue[image,{xmin;;xmax,ymin;;ymax}] gives the values of pixels whose coordinates range from {xmin,ymin} to {xmax,ymax}.
  • PixelValue[image,{x,y},"type"] gives the pixel value converted to the specified type.

Attributes: Protected, ReadProtected

Options: Padding
