DifferentiatorFilter

  • DifferentiatorFilter[data, ω c] applies a differentiator filter with a cutoff frequency ω c to an array of data.
  • DifferentiatorFilter[data, ω c,n] uses a filter kernel of length n.
  • DifferentiatorFilter[data, ω c,n,wfun] applies a smoothing window wfun to the filter kernel.

Attributes: Protected, ReadProtected

Options: Padding, SampleRate
