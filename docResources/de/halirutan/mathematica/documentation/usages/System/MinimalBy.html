MinimalBy

  • MinimalBy[{e1,e2, ... },f] returns a list of the ei for which the value of f[ei] is minimal.
  • MinimalBy[{e1,e2, ... },f,n] returns a list of the ei corresponding to the n smallest f[ei].
  • MinimalBy[f] represents an operator form of MinimalBy that can be applied to an expression.

Attributes:Protected, ReadProtected

Symbol has no options.
