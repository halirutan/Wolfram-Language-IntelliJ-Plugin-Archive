Row

  • Row[{expr1,expr2, … }] is an object that formats with the expri arranged in a row, potentially extending over several lines.
  • Row[list,s] inserts s as a separator between successive elements.

Attributes: Protected, ReadProtected

Options: Alignment, Background, BaselinePosition, BaseStyle, DefaultBaseStyle, Editable, Frame, FrameMargins, FrameStyle, ImageMargins, ImageSize, RoundingRadius, Selectable, StripOnInput, SyntaxForm
