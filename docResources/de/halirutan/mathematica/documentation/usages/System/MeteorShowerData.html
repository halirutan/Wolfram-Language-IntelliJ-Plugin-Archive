MeteorShowerData

  • MeteorShowerData["name","property"] gives the value of the specified property for the meteor shower "name".
  • MeteorShowerData[{name1,name2, ... },"property"] gives a list of property values for the specified meteor shower names.
  • MeteorShowerData["name", "property", annotation] gives the specified annotation associated with the property.

Attributes:Protected, ReadProtected

Symbol has no options.
