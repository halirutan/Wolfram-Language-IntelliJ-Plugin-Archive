PaneSelector

  • PaneSelector[{v1->expr1,v2->expr2, … },v] represents an object that displays as a pane containing the expri for which vi is equal to v.

Attributes: Protected, ReadProtected

Options: Alignment, AutoAction, Background, BaselinePosition, BaseStyle, ContentPadding, ContinuousAction, DefaultBaseStyle, FrameMargins, ImageMargins, ImageSize
