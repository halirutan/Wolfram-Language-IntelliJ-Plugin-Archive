ParametricNDSolve

  • ParametricNDSolve[eqns,u,{t,tmin,tmax},pars] finds a numerical solution to the ordinary differential equations eqns for the function u with the independent variable t in the range tmin to tmax with parameters pars.
  • ParametricNDSolve[eqns,u,{t,tmin,tmax},{x,xmin,xmax},pars] finds a numerical solution to the partial differential equations eqns.
  • ParametricNDSolve[eqns,{u1,u2, … },{t,tmin,tmax},pars] finds numerical solutions for the functions   ui.

Attributes: Protected

Options: AccuracyGoal, Compiled, DependentVariables, DiscreteVariables, EvaluationMonitor, InterpolationOrder, MaxStepFraction, MaxSteps, MaxStepSize, Method, NormFunction, PrecisionGoal, StartingStepSize, StepMonitor, WorkingPrecision
