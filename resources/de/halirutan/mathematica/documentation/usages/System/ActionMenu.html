ActionMenu

  • ActionMenu[name,{lbl1:>act1,lbl2:>act2, … }] represents an action menu with label name and with items labeled lbli that evaluates the expression acti if the corresponding item is chosen.

Attributes: Protected, ReadProtected

Options: Alignment, Appearance, AutoAction, Background, BaselinePosition, BaseStyle, ContentPadding, DefaultBaseStyle, DefaultMenuStyle, Enabled, Evaluator, FieldSize, FrameMargins, ImageMargins, ImageSize, MenuAppearance, MenuStyle, Method
