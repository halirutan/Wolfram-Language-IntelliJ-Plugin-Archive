StateFeedbackGains

  • StateFeedbackGains[ssm,{p1,p2, … ,pn}] gives the state feedback gain matrix for the state-space model ssm, such that the poles of the closed-loop system are pi.
  • StateFeedbackGains[{ssm,{in1, … }}, … ] specifies control inputs ini to use.

Attributes: Protected, ReadProtected

Options: Method, Tolerance
