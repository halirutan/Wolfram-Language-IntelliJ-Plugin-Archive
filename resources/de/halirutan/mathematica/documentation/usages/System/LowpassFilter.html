LowpassFilter

  • LowpassFilter[data, ω c] applies a lowpass filter with a cutoff frequency ω c to an array of data.
  • LowpassFilter[data, ω c,n] uses a filter kernel of length n.
  • LowpassFilter[data, ω c,n,wfun] applies a smoothing window wfun to the filter kernel.

Attributes: Protected, ReadProtected

Options: Padding, SampleRate
