GradientOrientationFilter

  • GradientOrientationFilter[image,r] gives an image corresponding to the local orientation parallel to the gradient of image, computed using discrete derivatives of a Gaussian of pixel radius r, returning values between -( π )/(2) and ( π )/(2).
  • GradientOrientationFilter[image,{r, σ }] uses a Gaussian with standard deviation σ .
  • GradientOrientationFilter[data, … ] applies orientation filtering to an array of data.

Attributes: Protected, ReadProtected

Options: UndefinedOrientation, Method, Padding, WorkingPrecision
