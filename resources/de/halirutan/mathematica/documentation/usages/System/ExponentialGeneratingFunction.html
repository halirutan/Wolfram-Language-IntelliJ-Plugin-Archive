ExponentialGeneratingFunction

  • ExponentialGeneratingFunction[expr,n,x] gives the exponential generating function in x for the sequence whose nth term is given by the expression expr.
  • ExponentialGeneratingFunction[expr,{n1,n2, … },{x1,x2, … }] gives the multidimensional exponential generating function in x1, x2, … whose n1, n2, … term is given by expr.

Attributes: Protected, ReadProtected

Options: Assumptions, GenerateConditions, Method, VerifyConvergence
