NDSolveValue

  • NDSolveValue[eqns,expr,{x,xmin,xmax}] gives the value of expr with functions determined by a numerical solution to the ordinary differential equations eqns with the independent variable x in the range xmin to xmax.
  • NDSolveValue[eqns,expr,{x,xmin,xmax},{t,tmin,tmax}] uses a numerical solution to the partial differential equations eqns.

Attributes: Protected

Options: AccuracyGoal, Compiled, DependentVariables, DiscreteVariables, EvaluationMonitor, InterpolationOrder, MaxStepFraction, MaxSteps, MaxStepSize, Method, NormFunction, PrecisionGoal, StartingStepSize, StepMonitor, WorkingPrecision
