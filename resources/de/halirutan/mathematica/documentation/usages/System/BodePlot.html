BodePlot

  • BodePlot[lsys] generates a Bode plot of a linear time-invariant system lsys.
  • BodePlot[lsys,{ ω min, ω max}] plots for the frequency range ω min to ω max.
  • BodePlot[expr,{ ω , ω min, ω max}] plots expr using the variable ω .
