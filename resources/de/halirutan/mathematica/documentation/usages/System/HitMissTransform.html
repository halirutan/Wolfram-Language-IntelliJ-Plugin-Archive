HitMissTransform

  • HitMissTransform[image,ker] gives the hit-and-miss transform of image with respect to the composite structuring element ker.
  • HitMissTransform[image,{ker1,ker2, … }] gives the union of the hit-and-miss transforms for all the structuring elements keri.
  • HitMissTransform[image,{ker1,ker2, … },t] treats values above t as foreground.

Attributes: Protected, ReadProtected

Options: Padding
