StateResponse

  • StateResponse[ssm,u,{t,tmin,tmax}] gives the numeric state response of the StateSpaceModel ssm to input u for tmin ≤ t ≤ tmax.
  • StateResponse[ssm,{u[0],u[1], … }] gives the response of the discrete-time state-space model ssm to the input sequence u[i].
  • StateResponse[ssm,u,t] gives the symbolic state response as a function of time t.
  • StateResponse[ssm,{u1, … ,um}, … ] gives the state response for multiple inputs ui.
  • StateResponse[{ssm,{x10,x20, … ,xn0}}, … , … ] gives the response with initial states xi0.
