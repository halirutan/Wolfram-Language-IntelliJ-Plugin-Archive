BandstopFilter

  • BandstopFilter[data,{ ω 1, ω 2}] applies a bandstop filter with cutoff frequencies ω 1 and ω 2 to an array of data.
  • BandstopFilter[data,{ ω 1, ω 2},n] uses a filter kernel of length n.
  • BandstopFilter[data,{ ω 1, ω 2},n,wfun] applies a smoothing window wfun to the filter kernel.

Attributes: Protected, ReadProtected

Options: Padding, SampleRate
