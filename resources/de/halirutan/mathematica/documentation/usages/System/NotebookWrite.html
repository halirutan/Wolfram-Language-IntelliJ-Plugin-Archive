NotebookWrite

  • NotebookWrite[notebook,data] writes data into a notebook at the current selection, setting the current selection to be just after the data written.
  • NotebookWrite[cell,data] replaces the given cell instead of the current selection.
  • NotebookWrite[notebook,data,sel] writes data into a notebook, setting the current selection to be as specified by sel.

Attributes: Protected, ReadProtected

Options: AutoScroll
