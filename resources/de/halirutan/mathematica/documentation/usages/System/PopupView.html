PopupView

  • PopupView[{expr1,expr2, … }] represents an object which displays as a popup menu whose items are the expri.
  • PopupView[{expr1,expr2, … },i] makes the ith entry be the one currently chosen.
  • PopupView[{expr1,expr2, … },i,base] displays as base if it is not being clicked.

Attributes: Protected, ReadProtected

Options: Alignment, Appearance, AutoAction, Background, BaselinePosition, BaseStyle, ContentPadding, ContinuousAction, DefaultBaseStyle, DefaultMenuStyle, Enabled, FieldSize, FrameMargins, ImageMargins, ImageSize, MenuAppearance, MenuStyle
