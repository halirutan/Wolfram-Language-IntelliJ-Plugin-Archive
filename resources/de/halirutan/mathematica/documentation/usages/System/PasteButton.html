PasteButton

  • PasteButton[expr] represents a button that pastes expr whenever it is pressed.
  • PasteButton[label,expr] displays with label on the button.

Attributes: Protected, ReadProtected

Options: Active, Alignment, Appearance, AutoAction, Background, BaselinePosition, BaseStyle, ContentPadding, DefaultBaseStyle, Enabled, Evaluator, FrameMargins, ImageMargins, ImageSize, Method
