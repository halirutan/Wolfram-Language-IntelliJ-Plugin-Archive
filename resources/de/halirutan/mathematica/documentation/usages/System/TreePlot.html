TreePlot

  • TreePlot[{vi ​ 1->vj ​ 1,vi ​ 2->vj ​ 2, … }] generates a tree plot of the graph in which vertex vik is connected to vertex vjk.
  • TreePlot[{{vi ​ 1->vj ​ 1,lbl1}, … }] associates labels lblk with edges in the graph.
  • TreePlot[g,pos] places roots of trees in the plot at position pos.
  • TreePlot[g,pos,vk] uses vertex vk as the root node in the tree plot.
