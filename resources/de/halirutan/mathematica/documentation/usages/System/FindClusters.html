FindClusters

  • FindClusters[{e1,e2, … }] partitions the ei into clusters of similar elements.
  • FindClusters[{e1->v1,e2->v2, … }] returns the vi corresponding to the ei in each cluster.
  • FindClusters[{e1,e2, … }->{v1,v2, … }] gives the same result.
  • FindClusters[{e1,e2, … },n] partitions the ei into exactly n clusters.

Attributes: Protected

Options: DistanceFunction, Method
